📚 Book Notes App (MVP)
Description

A simple full-stack web application to track books you’ve read.

Users can:

Add a book (title, author, ISBN, rating, notes, date read)

View all saved books

Delete books

Automatically display book covers using ISBN (OpenLibrary Covers API)

The app uses Express.js for routing, EJS for server-side rendering, and PostgreSQL for data storage.

Tech Stack

Node.js

Express.js

PostgreSQL

EJS

CSS
